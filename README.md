# FlipRadio Write Skill

Two skills for FlipRadio critical writing style — one for writing from scratch, one for polishing existing drafts.

## Skills

| Skill | Trigger | Purpose |
|-------|---------|---------|
| `flipradio-write` | `/flipradio-write`, 写文章, 翻转电台风格写作 | Write an article from scratch |
| `flipradio-polish` | `/flipradio-polish`, 润色, 帮我改改 | Polish an existing article |

## Install

```bash
ln -sf $(pwd)/flipradio-write ~/.claude/skills/flipradio-write
ln -sf $(pwd)/flipradio-polish ~/.claude/skills/flipradio-polish
```
